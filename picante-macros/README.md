# picante-macros

Procedural macros for `picante`.

This crate provides the macros that back Picante's query and database definitions, including:

- `#[input]`
- `#[interned]`
- `#[tracked]`
- `#[db]`

Most users should depend on [`picante`](https://docs.rs/picante) directly rather than using this crate on its own.
