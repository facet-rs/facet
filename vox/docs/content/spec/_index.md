# Vox Specification

The Vox specification defines the protocol and runtime model across layers:

- Requests and channels
- Connections and lanes
- Transport prologue and the bare conduit
- Conduit serialization behavior
- Failure and shutdown boundaries
- Link transports (stream and WebSocket)
